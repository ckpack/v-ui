@ckapck/v-ui

A Vue3.x component library

Modern, fully tree-shakable, theme switching, easy to use, and more than that

Document
View on GitHub

Modernization

🚀 Based on Vite, VUE3.x, TypeScript and other modern tools development

Separate UI from CSS

🎨 The component's UI, state logic and CSS are separated, and the theme style is switched through Hooks, CSS-IN-JS and CSS variables

Fully Customizable

🔨 Define component prefixes, CSS variable prefixes, and CSS class prefixes through configuration to avoid component name or style conflicts

Fully Shakeable Tree

💪 All components support on-demand loading by default
