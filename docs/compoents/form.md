## API

### button 属性

属性名   | 说明      | 类型        | 默认值   |
| ----- | ------- | --------- | ----- |
| theme  | 主题色      | `enum`  | default   |
| size | 按钮大小 | `enum` | medium
| round | 是否为圆形按钮 | `boolean` | false
| disabled | 是否为禁用状态 | `boolean` | false
| text | 是否为文本按钮 | `boolean` | false
| outlined | 是否为边框按钮 | `boolean` | false
| raised | 是否为凸起按钮 | `boolean` | false

### button 插槽

| 插槽名 | 描述     |
| :----- | :------- |
| default | 默认插槽 |
