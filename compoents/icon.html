Icon ​

这是一个图标组件, 你可以通过向其传递一个svg字符串或者symbol使用。

TIP

它的原理很简单，通过以下css设置svg的初始样式。

  • 这允许通过font-size控制svg大小
css
svg {
  width: '1em';
  height: '1em';
}
  • 这允许通过color控制svg颜色
css
svg {
  fill: 'currentcolor'; // 所以对于多色图标颜色相关属性将无效。
}
